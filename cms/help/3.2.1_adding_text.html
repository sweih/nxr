Adding text

  1. Choose Content in the main menu and then Website.


  2. Click the name of the page where you want to add text. If it is a sub-page, click the + -sign next to the name of the respective main page. When the sub page names are displayed, click the name of the page where you want to add text.

    Backoffice Interface is displayed.


  3. Edit, add, or delete the body text in Body.

    Click Edit button on the left of Body to open the editing screen. Text editing screen is displayed.  


  4. Write text on the blank area.

    You can format the text you have written with Rich Text Editor tools. Hover the mouse cursor over an icon to see its purpose.


  5. When you have finished, click Save and Back to go back to Backoffice Interface. If you click Back, you go back to Backoffice Interface but the changes made to the text will not be saved to the database.


  6. You can also add, delete or edit text in Headline. Click Edit button on the right.


  7. In Backoffice Interface preview the page by clicking Preview Page.
