Adding tables

  1. Choose Content in the main menu and then Website.


  2. Click the name of the page where you want to add a table. If it is a sub-page, click the + -sign next to the name of the respective main page. When the sub page names are displayed, click the name of the page where you want to add a table.

    Backoffice Interface is displayed.


  3. Add tables to Body. Click Edit button on the left of Body to open the editing screen.

    Text editing screen appears. Move the cursor to the point in the text where you want to add the table.


  4. Click "Insert/Edit Table" icon. "Insert Table -- Web Page Dialog" box is displayed.


  5. Define the values for the table:

    Number of rows in "Rows"
    Number of columns in "Columns"
    Width of the outer borders in "Border size"
    Alignment in the drop-down menu
    Width of the table in "Width"
    Height of the table in "Height"
    Space in between the cells in "Cell spacing"
    Space between the cells and the text in "Cell padding"
    Title for the table in "Caption" (optional)



  6. Click OK.


  7. You can move the table around in the text by clicking the right mouse button over it and holding the button down until you reach the desired position for the table.


  8. When you have finished, click Save and Back to go back to Backoffice Interface. If you Back, you go back to Backoffice Interface but the changes made will not be saved to the database.


  9. In Backoffice Interface preview the page by clicking Preview Page.
