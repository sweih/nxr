Creating metadata for the page

Metadata is information about the contents of the site. Some metadata is written into the HTML code and other metadata might be visible as alt-texts on images. You can add metadata for your page and the objects on your page.

To edit META-data, choose Content in the main menu and then Website.

  1. Choose the desired page you want to edit.
  2. Click the Edit META-Data button.

Three fields are displayed: Description box, Keywords and User profile.

  1. Type in the Description field a short description of the content as you would like to be displayed in search engine results.
  2. Type in the Keywords field information you find essential for your audience. This information will also be used by search engines.
  3. Type in the Profile field the intended user of the page (an editor, a developer, an administrator, etc.).

For example, if you are creating a page about a new product, type in the Description box a short description of the product as you would like it to be displayed in search engine results. In the Keywords box, indicate the product's type, name and other features that you think are important. In the User profile field, type in the intended user of the page.

You can add metadata (alt-text) for images while you are Adding images or when you already have an image in the Library. The alt-texts are shown when the cursor is on the image.

  1. To add metadata for images choose Library in the Content menu.

    2. The Content Library is displayed. The subfolders are displayed on the left-hand side.

  2. Click Fotos.

    3. The contents of the Fotos -folder is displayed.

    Note: You may also have images in the Layout-Data -folder.

  3. Click the Edit button next to the desired image.
  4. In the ALT-Tag field, insert the text you want to appear over the image.
  5. Click the Save and back button.

To add an alt-text to a text link, use the Internet text editor (such as FCKeditor). In N/X, you can find this by clicking on the Edit All button in the Edit Content menu. (Browse Sitemap>page>Edit Content>Edit All>Body)

Note: If you do not have an Internet text editor installed, you must insert the html tag manually.
