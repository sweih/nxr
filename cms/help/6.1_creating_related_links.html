Creating related links

The pages on your site are typically accessible from a menu. To enchance the navigation experience further, you can create links that take the user directly from one page to another. This type of links are called related links.

To create links:

  1. From the Content menu, click the Website command.

    2. The Browse sitemap area is displayed. On the site map you can see the pages currently on your site.

  2. Click the page where you want to place the related link.

    3. That page's properties are displayed on the right side of the screen.

  3. Go to the Content tab and click the Create Instances button on the Related Links area.
  4. Type the number of related links you want to create into the box and click the Create instances button.

    5. A new related link instance is created. To create more instances, repeat the step.

  5. Click the Edit button on the instance you just created.

    6. The related link instance is opened for editing. Fill in the following fields:

    Field Description
    Label Text to be displayed on the link. Typically this is the title of the link's target page.
    Internal Address The page you want to link to. If you are creating a link within your site, select the target page from the list.
    External Address Alternatively you can type a direct URL of the target page here. In this case leave the Internal Address box empty as this URL will override the internal address.
    Target

    The target attribute controls where the new page will be displayed when the user follows a link. Most of the time, clicking on a link simply loads a new page in the same window where the link was. However, with target, you can have the new page open in a new window, or if you are using frames, in another frame. You can target a frame by a name you have assigned it.

    Target also has four predefined values, which can be used as if certain windows and frames already have names without you having to assign them:

    • _blank
    • _parent
    • _self
    • _top

  6. Click the Save and Back button.

A related link is now created on the page. To verify the result, launch the page and look at your development site before publishing the live site.
