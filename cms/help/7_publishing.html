Publishing a page

Please note: Only authorised users can publish (launch) pages. If you have no authority to do this, please contact the Administrator in your organisation in order to publish pages.

Prerequisites for publishing a page:

  • N/X is installed.
  • Templates for a page are created.

1. Log in to N/X.

2. Choose Content tab in the upper left hand corner.

3. Choose Website on the menu. A "Browse Sitemap" menu appears.

4. Click the name of the page you want to edit. If you want to edit a sub-page, click the + -sign next to the name of the respective main page. When the sub page names appear, click the name of the page you want to edit.

5. A Backoffice Interface appears. Now you can edit any part of the page you choose. Click Edit button on the left to open the editing screen. See the Creating a homepage topic.

6. When you have finished editing, go back to the Backoffice Interface by clicking Save and Back. If you click Back, the changes you made to the content will not be saved.

7. Click Launch page. The contents that you edited now appear online.

8. Log out by clicking the Logout button on the top right hand corner.
