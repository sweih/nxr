Linking to external Web pages: Introduction

You should create links between the pages you have created, and link them to your partners' pages. Linking to external Web pages helps your readers find your partners.

Prerequisites: At this stage you need to have a web page with some content, and you need to have the exact and correct URL of the page you want to link to.

Note: You need permission to link to other companies' pages if you intend to link to them in business purposes.

  1. Select Content on top of the N/X working area. Choose Website.

  2. Select the page you want the link to reside on.
    3. At this stage the Page editing area is displayed.
  3. Choose the part of the page you want the link reside on. Usually the links are in the body text. Click the Edit icon on the right.
    4. At this stage the html editor is displayed.
  4. Highlight the text or image you want to choose as the link text.
  5. Click the linking button . A linking dialog box is displayed.

  6. In the space reserved for the URL, write the whole URL address. N.B. Check that the URL is valid and correct.
  7. Choose the way the link will be showed in the Target drop menu
    • Using "New Window (_blank)" causes the link to open in a totally new browser window, leaving the page with the refering link still open behind it.
    • Using "Same Window (_self)" loads the page within the same frame as the link tag.
    • Using "Topmost Window (_top)" within a link tag causes the new page to load in the full body of the window, which is useful if you ever want to break out of the frameset you have created and have a frameless page.
    • Using "Parent Window (_parent)" is similar to Topmost window, but will refer to the immediate parent of a frame. In more advanced frame usage there may be several nested frames and this allows more control over which frames are specified.
  8. Click "OK".
