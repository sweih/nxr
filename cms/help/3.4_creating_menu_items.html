Creating menu items

If you want the pages on your site to be accessible from a menu, you need to create the menu items and link them to the pages. This allows users to navigate the site more easily.

To create a menu item for a page:

  1. In the Content menu, click the Website command.

    2. The Browse sitemap area is displayed. On the site map you can see the pages currently on your site.

  2. Click the page you want to create a menu item for.

    3. The page's properties are displayed on the right side of the screen.

  3. Go to the Menu tab and fill in the following boxes:

    4. Box Description
    Name on Menu The name that you want displayed on the menu. Typically this name is the same as the title of the page you are linking to.
    Helptext for Menu A tooltip-like text displayed when the user places the mouse cursor on the menu item.
    Direct URL on Live Server An optional direct path to the page.

  4. Click the Save button.

A menu item is now created for your page. To verify the result, launch the page and look at your development site before publishing the live site.
