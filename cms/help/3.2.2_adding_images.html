Adding images

Before you add images to the page, you must upload them to Library.


3.2.2.1. Uploading images to Library


  1. Choose Content in the main menu and then Library.


  2. Click New Content. New Object screen appears.
    • Type the name of the object to Object Name field.
    • Choose the type of the object (Image) in the menu.
    • Type a description of the image in Description field (optional).
    • Type keywords for a search engine in Keywords field (optional).


  3. Click Create.

    The uploaded image appears on Content Library screen.



3.2.2.2. Adding images to a page

  1. Choose Content in the main menu and then Website.


  2. Click the name of the page where you want to add the image. If it is a sub-page, click the + -sign next to the name of the respective main page. When the sub page names are displayed, click the name of the page where you want to add the image.

    Backoffice Interface is displayed.


  3. Add images to Body. Click Edit button on the left of Body to open the editing screen.

    Text editing screen is displayed. Move the cursor to the point in the text where you want to add the image.


  4. Click Insert/Edit Image icon.

    "Image -- Web Page Dialog" box is displayed.

  5. Click Browse Server.

    Content Library window is displayed. Choose the image you want to add.



  6. Close Content Library window.


  7. Click OK in "Image -- Web Page Dialog" box. The image is displayed in the body text.


  8. You can move the image around in the text by clicking the right mouse button over it and holding the button down until you reach the desired position for the image.


  9. When you have finished, click Save and Back to go back to Backoffice Interface. By clicking Back, you go back to Backoffice Interface but the changes made will not be saved to the database.


  10. In Backoffice Interface preview the page by clicking Preview Page.
